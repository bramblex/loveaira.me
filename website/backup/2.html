▶什么是 PureScript

我们先看看 PureScript 官网怎么介绍自己的：

PureScript is a small strongly typed programming language that compiles to JavaScript.

正如官网所介绍的那样，PureScript 是一个强类型的并且编译到 JavaScript 的编程语言。从语法上来看，PureScript 是一个使用 Haskell 语法风格的强类型纯函数式语言。但是在语义上，PureScript 的作者称 PureScript 还是尊崇 JavaScript 的语义而不是 Haskell 语义。比如 PureScript 是立即求值，而 Haskell 是惰性求值等等。

PureScript 具备和 Haskell 类似的类型系统、类型推导、Type Class 等，但是同时也具有自己的特点。比如处理副作用的 Effect ，以及直接对应 JavaScript 对象的 Recrod 类型。然而最强的特点还是 PureScript 和 JavaScript 的互操作性了。

▶为什么选择 PureScript

一开始其实我并没有考虑用 PureScript， 甚至那时候我都还不知道 PureScript 是什么。我只是想用 Haskell 来装个逼，作为我的博客后端。然而我的 vps 实在是太破了，根本装不上 ghc 。所以只能退而求其次，最终找到了这么一个 PureScript。

PureScript 是一门通用的语言，所以不仅仅可以跑在 Node 上作为后端，也可以跑在浏览器上作为前端。所以决定干脆前后端都用 PureScript 。
